# nearornot - Proximity Monitoring Sensor Demo for iOS

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

This app demonstrates the use of the proximity monitoring sensor on iOS devices. When the device is held close to the user's face, the screen turns red, indicating that the device is near. When the device is held further away, the screen turns green, indicating that the device is far.

## Features
- Uses the proximity monitoring sensor to detect when the device is near the user's face
- Provides a visual indication of the proximity state (near or far) using color changes

## Requirements
- iOS 18.5 or later

## Usage
1. Clone the repository or download the source code.
2. Open the `nearornot.xcodeproj` file in Xcode.
3. Run the app on an iOS device or in the iOS Simulator.

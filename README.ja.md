# nearornot - iOSの近接センサーデモ
English README is here: [README.md](README.md)

iOSデバイスの近接センサーを使って、画面の色で近接状態を表示するシンプルなデモアプリです。

## 機能
- 近接センサーの監視
- 近接状態のリアルタイム表示（近い: 赤、遠い: 緑）

## 必要環境
- iOS 18.5以降
- Xcode 16.4以降

## 使い方
1. Xcodeでプロジェクトを開きます。
2. シミュレーターを使うか、実機でビルドしてインストールします。
3. デバイスの近接センサーの反応を体験できます。
